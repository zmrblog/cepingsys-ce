# 干部、部门考核测评系统 (CepingSys)

> 企业级考核测评系统，支持民主测评、干部考核、部门考核等多种场景

**[English README](README_EN.md)**

## 功能特性

### 📊 核心功能
- ✅ **测评管理** - 创建和管理考核测评任务
- ✅ **用户管理** - 用户信息管理和权限配置
- ✅ **部门管理** - 组织架构管理
- ✅ **指标管理** - 考核指标体系配置
- ✅ **数据统计** - 测评结果统计和分析
- ✅ **移动端** - 微信扫码登录参与测评
- ✅ **报表导出** - 测评结果导出

### 🏢 适用场景
- 党政机关干部考核
- 企事业单位绩效考核
- 部门年度测评
- 民主评议

## 快速开始

### 环境要求
- PHP 8.1+
- MySQL 5.7+
- Nginx / Apache

### 安装步骤
1. 下载源代码并上传到服务器
2. 访问 `http://your-domain/install.php`
3. 按照安装向导完成配置
